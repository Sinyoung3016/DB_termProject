# LibraryService

## 😺 프로젝트 소개

2021년 데이터베이스 수업의 TermProject로 진행한 도서관 서비스 입니다

(view 보다는 DB의 query문 사용에 집중하여 개발)

## 😺 사용 기술

front : `react`

DB server : `oracle DB`

## 😺 실행 방법

server 실행 후, client 실행

> npm install

> npm start
